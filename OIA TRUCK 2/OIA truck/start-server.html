🚀 派车单管理系统 - 动态网页版

✅ 动态网页功能已配置完成！

您的派车单管理系统已成功升级为动态网页版本，支持云端数据同步和离线工作模式。

📋 功能特性

  • 实时数据同步: 支持多设备间数据实时同步
  • 离线工作模式: 网络断开时自动切换到本地存储
  • 云端数据备份: 数据自动备份到Firebase云数据库
  • 网络状态指示: 实时显示在线/离线状态
  • 响应式设计: 适配各种设备屏幕

⚠️ Firebase配置提醒

要启用云端数据同步功能，请按以下步骤配置Firebase：

  1. 访问 Firebase Console
  2. 创建新项目并启用Firestore数据库
  3. 获取Web应用配置信息
  4. 编辑 js/modules/firebase.js 文件，替换配置信息

🛠️ 本地开发服务器（推荐）

为了获得最佳体验，建议安装Node.js并使用本地服务器：

方法一：安装Node.js

1. 下载并安装 Node.js: https://nodejs.org/
2. 重启命令提示符
3. 在项目目录运行: npm install
4. 启动开发服务器: npm run dev

方法二：使用Python（如果已安装）

# Python 3.x
python -m http.server 3000

# Python 2.x
python -m SimpleHTTPServer 3000

方法三：使用其他工具

  • Live Server (VS Code扩展)
  • XAMPP 或 WAMP
  • IIS Express

📱 部署到云端

将应用部署到Netlify等平台：

  1. 将项目上传到GitHub
  2. 连接到Netlify
  3. 配置环境变量
  4. 自动部署完成

详细步骤请参考 README.md 文件。
